引言

QuickLook 是什么？

QuickLook 是苹果自 OS X 10.5 版本引入的一项技术。用户可以在不打开文件的情况下，只需要选中文件，单击空格键，即可快速预览文件内容。

QuickLook 扩展性

查询苹果开发者文档关于 QuickLook 介绍部分：

QuickLook is a technology introduced in OS X version 10.5 that enables client applications, such as Spotlight and the Finder, to display thumbnail images and full-size previews of documents. For documents of common content types—notably HTML, RTF, plain text, TIFF, PNG, JPEG, PDF, DAE, and QuickTime movies—this support is automatic. However, applications with documents that are of less common or even private content types can still take advantage of the QuickLook feature. Those applications can include QuickLook generators: plug-ins that convert a given document from its native format into a format that QuickLook can display to users.

我们可以得知（简单翻译一下）：QuickLook 是苹果自 OS X 10.5 版本引入的一项可以让客户端应用程序（如：Spotlight、Finder）能够显示文件的缩略图和全尺寸预览的一项技术。对于常见的文档类型，例如：HTML、RTF、文本文件、TIFF、PNG、JPEG、PDF、DAE 和 QuickTime 视频是原生支持的。但是，不常见的文档类型抑或是私有内容类型仍可以利用 QuickLook 这一特性。只要这些应用程序包含可以将该文档的原生格式转化为 QuickLook 显示的格式的插件即可。

其原生支持一些常见的文件类型，而且给其他不支持类型的文件提供了扩展性支持。这就为我们对其他目前不支持的文件类型指明了道路的方向。

扩展

扩展途径

  1. 硬核方式：自行开发；
  2. 简单方式：找寻别人造好的轮子，为己所用。

当然我选择第二种，哈哈哈。

扩展方式

经过在 GitHub 上搜寻，找到了 quick-look-plugins 这一仓库，简直犹如发现宝藏一般。

根据文档介绍，需使用 Homebrew Cask 来安装，如果你还未安装 Homebrew ，可以点击这里来安装。

一行命令，安装目前所有的第三方扩展：

brew cask install qlcolorcode qlstephen qlmarkdown quicklook-json qlimagesize suspicious-package quicklookase qlvideo

我这里都已安装所以显示的为已安装。

扩展注意事项

安装完毕之后，如果你的 macOS 系统版本为最新的 Catalina 10.15 版本，需要先执行一下操作：

xattr -r ~/Library/QuickLook

查看刚刚安装的插件的文件属性，如果发现有 com.apple.quarantine 则还需执行以下命令：

xattr -d -r com.apple.quarantine ~/Library/QuickLook

扩展预览

注：以下题图均来自 quick-look-plugins 。

QLColorCode

brew cask install qlcolorcode 预览源码，同时支持代码高亮。

QLStephen

brew cask install qlstephen 预览没有文件扩展名的文本文件，例如：README，CHANGELOG，index.styl 等。

QLMarkdown

brew cask install qlmarkdown 预览 Markdown 文件。

QuickLookJSON

brew cask install quicklook-json 预览 Json 文件。

BetterZipQL

brew cask install betterzip 预览压缩文件。

qlImageSize

brew cask install qlimagesize 预览图片，并显示图片分辨率和大小。

Suspicious Package

brew cask install suspicious-package 预览安装包内内容。

QuickLookASE

brew cask install quicklookase 预览 Adobe ASE 文件。

QLVideo

brew cask install qlvideo 预览绝大多数类型视频文件。

其他扩展

ProvisionQL

brew cask install provisionql 预览 ipa 包内文件。

QuickLookAPK

brew cask install quicklookapk 预览 apk 包内文件。

quicklook-pat

brew cask install quicklook-pat 预览 Adobe PAT 文件。

WebP

brew cask install webpquicklook 预览 WebP 格式文件。
