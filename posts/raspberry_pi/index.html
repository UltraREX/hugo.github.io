什么是树莓派？

树莓派（英语：Raspberry Pi）是一款基于 Linux 的单片机电脑，由英国的树莓派基金会所开发，目的是以低价硬体及自由软体促进学校的基本计算机科学教育。

树莓派由两家拥有许可的公司（Element 14/Prmier Farnell 和 RS Components）生产，这两家公司都在网上出售树莓派。

树莓派每一代均使用博通（Broadcom）出产的 ARM 架构处理器，内存在 256MB 和 4GB 之间，主要使用 SD 卡或者 TF 卡作为存储媒躰，配备 USB 接口、HDMI 的视频输出（支持声音输出）和 RCA 端子输出，内置 Ethernet/WLAN/Bluetooth 网络链接的方式（依据型号决定），并且可使用多种操作系统。产品线型号分为 A 型、B 型、Zero 型和 ComputeModule 计算卡。

—— Wikipedia

生态

树莓派之所以受到全世界的青睐，就在于它小小的身体蕴含着巨大的能量，同时还有成熟的社区生态。它不仅可以作为欠发达地区或国家学生接触计算机的首选，还可以配合一些定制化的系统或者插件，摇身一变为智能家庭中枢、NAS、魔镜、家庭影院系统等等等等，不胜枚举。

「初始化」

所需物品清单

  1. 树莓派（型号根据需求）
  2. SD 卡
  3. 电脑

安装系统

下载系统

获取途径包括但不限于：官网。同时也有非常丰富的第三方系统用于特定的场景，详情可自行搜索，这里不再赘述。

目前官网提供两种安装系统的方式：

  1. Raspberry Pi Imager
  2. 手动安装

这里以 Raspbian （官方系统）为例，下载好后，得到一个名如 2020-02-13-raspbian-stretch.zip 的压缩包，解压得到名如 2020-02-13-raspbian-stretch.img 的镜像文件。

烧录系统

将 SD 卡插入电脑，打开烧录系统的软件。这里推荐使用 balenaEtcher ，只需要三步就可以完成系统的烧录。

  1. 选择所需要烧录的将镜像文件；
  2. 选择目标盘；
  3. 点击烧录，完成。

当系统烧录完成之后，将 SD 卡插入树莓派，即可启动树莓派了。

不过为了更好的使用树莓派，刚刚的 SD 卡，不要着急拔，我们还需要一些「善后」工作。

「善后」工作

开启 SSH

打开 SD 卡的根目录，创建名为 ssh 的文件即可，无需输入任何内容。

设置 Wi-Fi

如果没有有线环境或者不想使用网线来给树莓派提供网络，可以设置 Wi-Fi 信息，在树莓派启动的时候就会自动连接上你所设置的 Wi-Fi。

同样在 SD 卡的根目录，创建名为 wpa_supplicant.conf 的文件，文件内容为：

country=SE
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1

network={
    ssid="无线链接的 SSID"
    psk="无线密码"
}

其中需要在 ssid 和 psk 里填写真实的 Wi-Fi 信息。

总结

至此，你的树莓派可以安心启动了，不必担心没有有线网络、键盘、鼠标、显示器从而无法访问树莓派了。我们可以通过 SSH 来访问它，安装各种你需要的包或者设置环境了。

Don't Panic.​
