引言

Markdown 固然简洁优雅，但有时候我们还期望在文档中添加一些更加丰富的内容（视频、<iframe>，等等），就不得不写一些 HTML 在 Markdown 文档里，看起来极其不美观和优雅。那么有没有一种方式既可以像 Markdown 那样优雅又可以让我添加更多丰富内容的方法呢？答案是：有的 —— Shortcodes（我不知道如何翻译，短代码？似乎又比较难听，姑且就不翻译了）。

什么是 Shortcodes

Shortcodes are simple snippets inside your content files calling built-in or custom templates.

正如 Hugo 文档介绍这般，Shortcodes 是写在文档中的简单代码片段，形如：{{< youtube 4QOkcGwjVow >}} （如果需要在文档内显示 Shortcodes 语句，需要在标签内添加注释将语句包起来即可，{{</* somecode */>}}）一个简单的标签就可以在你的文档内插入 YouTube 的视频。

而如果是内嵌 HTML 的方式，你需要写这些 HTML 代码在你的文档里，高下立判。

<iframe width="560" height="315" src="https://www.youtube.com/embed/4QOkcGwjVow" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

编写自己的 Shortcodes

当然你也可以根据自己的需求来编写自己的 Shortcodes 模板，操作起来也非常简单。

创建 Shortcodes 模板

在 layouts/shortcodes 目录下创建模板文件（如果 layouts 文件夹下没有 shortcodes 文件夹则需创建），模板文件名即为标签 {{< somecode >}} 内的命令名。

详细的语法这里也不再赘述了，详情请参见官方文档，讲述地特别清晰。

以 image 为例：

  • 创建 image.html 模板文件

  • 编写模板文件内容

    <p>
<div align="center">
    <img src="/img/loading.gif" data-src="{{ .Get `src` }}" {{ with .Get `alt` }}alt="{{.}}"{{ end }} class="lazyload" />
    {{ with .Get `title` }}<p style="color:gray;font-size:small;text-align:center;">- {{.}} -</p>{{ end }}
</div>
</p>

可以看到 image 这一 Shortcode 接收 src alt title 三个参数，参数的含义我想不用过多解释了。同时使用 lazyload 来「懒加载」图片资源，同时为了用户体验更好，添加了一个 Loading 资源，在图片资源没有加载完成时显示，如下。

  • 使用

在文档内编写这样的标签 {{< image src="the/image/path" alt="" title="" >}} 即可，其中 alt 和 title 可缺省。Hugo 在生成网页的时候，就会把该标签替换成对应 shortcodes 模板内的内容。
